💡 Gender Rules & Tips

👫 Feminine Formation

  • Most professions add -in for feminine
  • Lehrer → Lehrerin (Teacher)
  • Manager → Managerin (Manager)
  • Some change vowels: Arzt → Ärztin (Doctor)

🔢 Plural Rules

  • Feminine plurals typically add -innen
  • Lehrerin → Lehrerinnen
  • Masculine plurals vary by profession
  • Some stay the same: Manager → Manager

📝 Usage Tips

  • Always use correct article: der/die
  • In job applications, specify gender form
  • Modern German often uses both forms
  • Example: "Lehrer/Lehrerin gesucht"

🗣️ Pronunciation

  • Stress usually on first syllable
  • -er ending sounds like "ehr"
  • -in ending pronounced "een"
